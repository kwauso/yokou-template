# README

## About
もともとのテンプレートは、[RGの予稿テンプレート](https://github.com/sfc-rg/yokou-template)です。それを編集し、LuaLaTeXでコンパイルできるようにしました。

## 環境
Macでの使用を想定しています。Latexの環境は[Tex Live](https://texwiki.texjp.org/?TeX%20Live)を使用しています。

## つかいかた
1. `resume.tex`をいじる
2. `lualatex resume.tex`
4. `make clean`
